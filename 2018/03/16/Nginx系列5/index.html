Nginx配置文件详解

Nginx配置分为各个配置块，主配置块负责全局配置，子配置块可以继承全局配置，也可以相应的配置不同设置。

  1. 运行必备的配置
  2. 优化性能的配置
  3. 调试及定位问题的配置
  4. 事件驱动相关配置

http协议配置块位于总体配置块中

  1. 与套结字相关的配置
  2. 定义路径相关的配置
  3. 定义客户端请求的相关配置
  4. 对客户端进行限制的相关配置
